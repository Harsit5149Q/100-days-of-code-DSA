### Search in Step Array

Problem Statement
You are given a step array arr[], an integer k representing the maximum allowed difference between adjacent elements, and an integer x representing the key to be searched. A step array is defined as an array where the absolute difference between any two adjacent elements is at most k. Your task is to find the index of the first occurrence of x in the array. If x is not present, return -1.

Input Format
An integer array arr[].
An integer k representing the maximum adjacent difference.
An integer x representing the element to be searched.

Output Format
Print the index of the first occurrence of x in the array. If x is not found, print -1.

Sample Input
arr[] = [4, 5, 6, 7, 6]
k = 1
x = 6

Sample Output
2

Explanation
The element 6 first appears at index 2 in the array, so the output is 2.
