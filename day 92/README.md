### Quick Sort Implementation

Problem Statement
Implement the Quick Sort algorithm to sort an array of integers using a pivot-based partitioning strategy.

Input Format
n
n space-separated integers

Output Format
Print the sorted array.

Sample Input
5
64 34 25 12 22

Sample Output
12 22 25 34 64

Explanation
Elements are partitioned around a pivot such that smaller elements go to the left and larger to the right, recursively sorting subarrays.
