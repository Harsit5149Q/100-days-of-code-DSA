## Question 1

### Meeting Rooms Scheduling

Problem Statement
Given a list of meeting time intervals, determine the minimum number of meeting rooms required so that all meetings can be held without overlap.

Input Format
n
n pairs of start and end times

Output Format
Print the minimum number of rooms required.

Sample Input
3
0 30
5 10
15 20

Sample Output
2

Explanation
Two meetings overlap, so at least two rooms are required.
