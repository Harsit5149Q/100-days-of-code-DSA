### Counting Inversions in Array

Problem Statement
Given an array of integers, count the number of inversion pairs.
An inversion is a pair (i, j) such that i < j and arr[i] > arr[j].

Input Format
n
n space-separated integers

Output Format
Print the number of inversions.

Sample Input
5
2 4 1 3 5

Sample Output
3

Explanation
The inversion pairs are (2,1), (4,1), and (4,3).
