### Bucket Sort (Float Values 0–1)

Problem Statement
Implement Bucket Sort to sort floating-point numbers uniformly distributed in the range [0, 1).

Input Format
n
n space-separated floating-point numbers

Output Format
Print the sorted array.

Sample Input
5
0.78 0.17 0.39 0.26 0.72

Sample Output
0.17 0.26 0.39 0.72 0.78

Explanation
Elements are distributed into buckets, each bucket is sorted individually, and then concatenated.
