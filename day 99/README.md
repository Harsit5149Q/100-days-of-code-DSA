### Find Peak Element in Array

Problem Statement
Given an array of integers, find an index of a peak element. An element is a peak if it is greater than its neighbors.

Input Format
n
n space-separated integers

Output Format
Print the index or value of a peak element.

Sample Input
6
1 2 3 1 5 6

Sample Output
2

Explanation
Element 3 at index 2 is greater than its neighbors.
